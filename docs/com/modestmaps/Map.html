public function Map(width:Number = 320, height:Number = 240, draggable:Boolean = true, mapProvider:IMapProvider = null, ... rest)

Initialize the map: set properties, add a tile grid, draw it. Default extent covers the entire globe, (+/-85, +/-180).

Parameters
width:Number (default = 320) — Width of map, in pixels.
 
height:Number (default = 240) — Height of map, in pixels.
 
draggable:Boolean (default = true) — Whether the map can be dragged or not.
 
mapProvider:IMapProvider (default = null) — Desired map provider, e.g. Blue Marble.
 
... rest — Either a MapExtent or a Location and zoom (comma separated)

zoomBy()method 
protected function zoomBy(dir:int):void

Adds dir to grid.zoomLevel, and rounds up or down to the nearest whole number. Used internally by zoomIn and zoomOut (keeping it DRY, as they say) and overridden by TweenMap for animation.

grid.zoomLevel calls the grid.scale setter for us which will call grid.prepareForZooming if we didn't already and grid.doneZooming after modifying the zoom level.

Animating/tweening grid.scale fires START_ZOOMING, and STOP_ZOOMING MapEvents unless you call grid.prepareForZooming first. Be sure to also call grid.stopZooming at the end of your animation. Parameters

dir:int — the direction of zoom, generally 1 for zooming in, or -1 for zooming out
