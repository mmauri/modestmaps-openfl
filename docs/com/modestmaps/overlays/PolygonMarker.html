public function PolygonMarker(map:Map, locations:Array, autoClose:Boolean = true)

Creates a polygon from the given array (or array of arrays) of Locations. The polygon will use the given map to project the locations, and should be added to an instance of PolygonClip, which will add and remove it from the stage and position it as required. If an array of arrays of Locations is given, the first array will be drawn as the outer ring of the polygon, and subsequent arrays will be treated as holes if they overlap it.
