Packagecom.modestmaps.overlays
Classpublic class PolylineClip
InheritancePolylineClip flash.display.Sprite

Polyline clip for rendering Polyline instances on your map. Polylines can be added using: 
   var polylineClip:PolylineClip = new PolylineClip(map);
   map.addChild(polylineClip);
   
   var polyline:Polyline = new Polyline('poly-id-1', [ new Location(10,10), new Location (20,20) ]);
   polylineClip.addPolyline(polyline);
